总体思路

Fly 模块#

实现 端口监听、启动服务器，开放 路由配置、跨域设置、静态服务的接口配置。

requestParser#

解析http报文

Router#

实现路由功能，用于分发http请求，实现路由接口。

Handler#

实现http处理接口，用与编写具体业务逻辑

CGI (可选)#

使用child.exec函数实现进程多开，多语言CGI。

Response#

拼接http报文
